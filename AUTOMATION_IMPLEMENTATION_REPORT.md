## 🚀 使用方法

### 启动自动化循环

```bash
# 基本用法
zhushou autonomy start-genesis-loop

# 自定义配置
zhushou autonomy start-genesis-loop \
  --scan-interval 15 \
  --max-concurrent 5 \
  --workspace /path/to/workspace
```

### 查看状态

```bash
zhushou autonomy status
```

### 访问前端面板

```bash
# 启动网关
zhushou gateway

# 访问治理层面板
open http://localhost:3000/governance
```

- `GenesisTeamLoop`: 独立的自动化循环引擎
- `SandboxReplayRunner`: 独立的重放执行器
- `PostPromotionObserver`: 独立的观察器

### 2. 类型安全

所有模块都使用 TypeScript 编写，提供完整的类型定义：
- 接口定义清晰
- 类型推导准确
- 编译时检查

### 3. 可扩展性

所有配置都可外部化，支持灵活定制：
- 扫描间隔可配置
- 并发数可配置
- 阈值可配置
- 行为可定制

### 4. 可观测性

完善的日志和监控：
- 结构化日志输出
- 关键指标收集
- 状态实时可见

### 5. 容错设计
