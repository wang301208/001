## Genesis Team 工作流程（自我组织能力）

```
┌─────────────────────────────────────────────────────────────────┐
│                    Genesis Team Loop                             │
│                  (Self-Organization Workflow)                    │
└─────────────────────────────────────────────────────────────────┘

    ┌──────────────┐
    │   Sentinel   │ ◄── 检测能力缺口
    │  Detection   │     输入: capability_inventory.asset_registry_sync_required
    └──────┬───────┘
           │ 输出: gap_signal, risk_summary
           ▼
    ┌──────────────┐
    │ Archaeologist│ ◄── 分析根本原因
    │   Analysis   │     输入: gap_signal
    └──────┬───────┘
           │ 输出: root_cause_map, change_plan
           ▼
    ┌──────────────┐
    │ TDD Developer│ ◄── 实现解决方案
    │ Implementation│    输入: root_cause_map
    └──────┬───────┘
           │ 输出: candidate_asset, test_delta
           ▼
    ┌──────────────┐
    │     QA       │ ◄── 验证和回归测试
    │   Replay     │     输入: candidate_asset
    └──────┬───────┘
           │ 输出: qa_report, replay_report
           ▼
    ┌──────────────┐
    │  Publisher   │ ◄── 决策晋升或回滚
    │   Decision   │     输入: qa_report
    └──────┬───────┘
           │ 输出: promotion_record, rollback_decision
           ▼
    ┌──────────────┐
    │  Librarian   │ ◄── 注册新资产
    │ Registration │     输入: promotion_record
    └──────┬───────┘
           │ 输出: asset_manifest, registry_update
           ▼
    ┌──────────────┐
    │  Asset Pool  │ ◄── 能力池更新
    └──────────────┘

✅ 能自我组织 - Genesis Team 自主形成组织结构和工作流程
```
